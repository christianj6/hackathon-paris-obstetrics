About this Project

This web application is a part of the F'AI'R Hackathon, which aims to develop innovative solutions to facilitate knowledge exchange between different institutions. The website leverages data from an obstetric training manual database and recommends content based on user proficiency ratings, natural language processing, and topic modeling techniques.
